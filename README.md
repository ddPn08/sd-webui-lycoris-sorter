## 使い方

- 入力  
    分類の対象が入っているフォルダのパスを入力。

- 出力  
    分類結果を出力するルートパス。  
    `C:\hoge` に指定したとすると、下記のような構成になる。  

    ```
    C:\hoge\
        lora\
            LoRa形式などの物
        lycoris\
            lycoris形式などの物
    ```

フォルダ構造は入力フォルダが継承されますが、現状では付随のサムネ画像やら情報ファイルは引き継がれません。置いてけぼりです。  
そういうこともあり、本システムは常用はせず、形式の混在した状態を真っ当な状態にするための、緊急対策的な物として使用ください。
